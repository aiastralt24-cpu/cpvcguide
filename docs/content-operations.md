# Content Operations Workflow

1. Brief
   - lock primary query
   - lock audience
   - lock unique angle
   - lock pillar and intent
2. Draft
   - answer query in first paragraph
   - fill required page sections
   - add metadata and related links
3. Review
   - technical accuracy pass
   - editorial tone pass
   - schema and linking pass
4. Publish
   - final metadata check
   - QA mobile and table rendering
5. Refresh
   - 90-day review
   - improve, consolidate, or retire
